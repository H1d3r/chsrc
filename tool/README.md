# 工具说明

1. `installer.sh`

    Bash 编写的一键安装工具

2. `installer.ps1`

    PowerShell 编写的一键安装工具

3. `download-pre-on-GitHub.ps1`

    从 GitHub `pre` release 上下载所有附件

<br>

## `rawstr4c`

C语言字符串生成工具，本身是为了本项目而开发，是放在本仓库下的一个完整子项目。但为了维护更方便，拆分到新的仓库: <https://github.com/RubyMetric/rawstr4c>
